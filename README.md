📘 Actividad
Título de la actividad:
Configuración del entorno backend con Node.js/npm y construcción del primer servidor con Express.js que responde peticiones HTTP.

Modalidad: Tarea individual

📝 Descripción
Construir un servidor Express.js que gestione una lista de estudiantes almacenada en un array de JavaScript (datos quemados, sin base de datos). El servidor debe responder correctamente a peticiones GET, POST, PUT y DELETE, devolviendo siempre JSON con el código de estado HTTP apropiado.

La temática para el API REST es la administración de estudiantes y debe incluir endpoints funcionales para consultar, agregar, actualizar y eliminar estudiantes.
